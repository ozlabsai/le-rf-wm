Debugging: CLS Token Problem

INVESTIGATION

CLS token std across batch: 0.000

Patch tokens std across batch: 0.847

Patches carry information. CLS does not.

ROOT CAUSE

CLS token at random init is a single shared learned parameter. Identical for every sample in the batch. Zero variance = zero information.

BEFORE

output = CLS_token # shared param
std(output) = 0 # collapsed

→

AFTER

output = mean(patches) # per-sample
std(output) = 0.847 # diverse

Mean pooling preserves per-sample patch diversity through the transformer. CLS token destroys it at initialization.

Fix: CLS token pooling → mean patch pooling. But this was only the first collapse mechanism.
