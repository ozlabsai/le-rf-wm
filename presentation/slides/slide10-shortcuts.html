Loss Functions Define Shortcuts

MSE LOSS

Allows shrinkage. Predict low-magnitude vectors near zero to minimize error without learning structure.

ABSOLUTE TARGETS

Allows memorization. Model learns scene identity (which scene am I in?) rather than temporal change.

OVERLAPPING SLICES

Allows copying. When context and target share timesteps, the predictor can echo shared frames.

L2-NORMALIZED MSE

Removes shrinkage. Forces directional accuracy regardless of magnitude.

RESIDUAL TARGETS (Δz)

Removes memorization. Must predict temporal change, not scene identity.

NON-OVERLAPPING

Removes copying. Target starts after context ends. No shared frames.

"Regularization cannot create signal — it can only preserve it."

Each fix removes a shortcut. The model is forced to learn actual dynamics.
